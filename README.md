## Bruin Market
An online website where UCLA students can trade their used goods with each other. 

### Technology Stack
<img src="Tech Stack Screenshot.png" width="550">

### Set Up
```
git clone https://github.com/MubaiHua/HOTH9.git
```

```
cd HOTH9/server
npm install
npm start
```

```
cd HOTH9/client
npm install
npm start
```
